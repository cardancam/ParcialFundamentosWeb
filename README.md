## Descripción del proyecto

Este proyecto consiste en la recreación de la página oficial de **ExpressVPN**.
El objetivo fue practicar la estructura de una página web moderna utilizando **HTML y CSS**, aplicando conceptos de maquetación, diseño responsive y uso de enlaces externos.


## Tecnologías utilizadas

Para el desarrollo del proyecto se utilizaron las siguientes tecnologías:

* **HTML5**

  * Estructura de la página
  * Uso de etiquetas semánticas como `header`, `section`, `footer`
  * Enlaces externos con `<a>`
  * Inserción de imágenes

* **CSS3**

  * Estilos de la página
  * Uso de **Flexbox** para la navegación
  * Uso de **Grid** para organizar elementos
  * Diseño **responsive** mediante `@media queries`
  * Efectos `hover` en botones y enlaces

## Funcionalidades implementadas

* Botones que redirigen a enlaces externos
* Íconos de dispositivos que funcionan como enlaces
* Diseño adaptado a diferentes tamaños de pantalla
* Organización de contenido en secciones
* Footer con enlaces de política de privacidad y términos de servicio

## Experiencia durante el desarrollo

Durante el desarrollo del proyecto se practicó la **estructura y organización de una página web**.
Se aprendió a combinar HTML y CSS para crear un diseño similar al de una página profesional.

También se reforzaron conocimientos como:

* Uso de **clases y estilos CSS**
* Posicionamiento de elementos
* Creación de botones interactivos
* Uso de Git para control de versiones
* Trabajo colaborativo mediante **ramas y Pull Requests**

## Control de versiones

El proyecto se gestionó utilizando **Git**, donde cada integrante trabajó en su propia rama.
Posteriormente se realizaron **Pull Requests** para integrar los cambios al repositorio principal.
